Detailed Description

template<class T>
class xl::CloneableUniquePtr< T, std::enable_if_t< std::is_same_v< std::result_of_t< decltype(&T::clone)(T)>, T * > > >

Same as std::unique_ptr (any differences are bugs/missing features) except copyable via clone method on contained value. Only allows contained values with clone() method defined
