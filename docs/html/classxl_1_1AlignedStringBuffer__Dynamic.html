Detailed Description

template<size_t alignment_v>
class xl::AlignedStringBuffer_Dynamic< alignment_v >

Dynamically-allocated buffer for AlignedString. Buffer will be aligned to the specified alignment and grows in multiples of the alignment

Template Parameters
alignment_valignment of the dynamically-allocated buffer Must be 16 or 64
