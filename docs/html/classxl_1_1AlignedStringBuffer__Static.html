Detailed Description

template<size_t size>
class xl::AlignedStringBuffer_Static< size >

An inline storage buffer for AlignedString which uses a fixed-length array for storage

Allows storage of a string of length size-1 and then a NUL terminator

Uses the last byte to store a representation of the length in such a way that the value is 0 when the string is full and therefor doubles as the NUL terminator

Template Parameters
sizemaximum amount of size the entire object can use, including any overhead
